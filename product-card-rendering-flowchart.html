Static (Manual) Advantages

  • Full HTML at request time — zero JS dependency
  • Fastest first contentful paint
  • SEO-friendly (crawlable product markup)
  • No layout shift (CLS = 0 for cards)
  • Works with JS disabled
  • Bracket notation resolved server-side

Dynamic (Recently Viewed) Advantages

  • Personalized per-visitor (localStorage data)
  • No server knowledge of browsing history needed
  • Targeting algorithms (AI, price sort, etc.)
  • Reactive to variant changes via Alpine
  • Can mix sources with fallback chains
  • Products fetched & cached in IndexedDB

Why It Matters

  • The same card metaobject powers both paths — the merchant designs the card once and the theme decides the render strategy based on the data source.
  • Static = "Shopify knows the products at request time"
  • Dynamic = "Only the browser knows the products"
